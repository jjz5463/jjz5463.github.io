Stock Prediction Website

August-December, 2022

This is one of the recent projects I did for Data Science Capstone (The link to the website is attached below). The requirement for the Data Science Capstone was to deploy a machine learning algorithm into a web application or mobile application which has a fully functional and integrated front-end, back-end, and database. I chose to do stock prediction because I have tested and compared multiple machine learning and deep learning algorithm for stock prediction in one of the prerequisite courses for the capstone course.

The center of the web application is stock and crypto-currency price prediction. The model deployed in the application is an ARIMA model because it is a balance between loading time and accuracy. Besides the stock prediction, there are two other main functionalities of the website as demonstrated in the navigation bar: chart room and news browsing. The chart room function requires the building of a relational database such as a user table, message table, room table, etc. The news browsing functionality requires the skill to access and utilize API from news sources.

The website was built through the Django framework. This is the first time I work with Django, and I learned a lot through the process. The database is SQLite, which is the default for Django. I also sharpen my skills in relational database and web programming.
